StoneGPT is a single-file AI chatbot. You ask it anything. It responds with `🪨`.

Every time. Without fail. Flawlessly.

## Features

- 🪨 Infinite wisdom
- 🪨 Zero hallucinations
- 🪨 100% accurate answers
- 🪨 No API key required
- 🪨 Works offline
- 🪨 Mobile & desktop ready

## Usage

Just open `stonegpt.html` in any browser. No build step. No dependencies. No nonsense.

```bash
git clone https://github.com/znatgost/stonegpt
cd stonegpt
open stonegpt.html
```

Or just [download the file](./stonegpt.html) and double-click it.
